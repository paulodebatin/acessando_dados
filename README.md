* mongodb-nodejs
* mongodb-python
* mongodb-c#

* elasticsearch-java
* elasticsearch-nodejs
* elasticsearch-python
* elasticsearch-c#

* redis-java
* redis-nodejs
* redis-python
* redis-c#

* consul-java
* consul-nodejs
* consul-python
* consul-c#
